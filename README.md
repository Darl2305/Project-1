# Portafolio - Ciencia de Datos y Machine Learning

Portafolio personal construido con Next.js (tema oscuro/claro) para mostrar tu enfoque en ciencia de datos, machine learning, analítica y visualización. Incluye secciones de “Sobre mí”, “Proyectos Personales” y “Contacto”.


1. Abrir [http://localhost:3000](http://localhost:3000)

## Deploy en Vercel

1. Sube el proyecto a GitHub
2. Conecta el repo en [vercel.com](https://vercel.com)
3. Vercel detectará Next.js automáticamente

## Personalización

- **Contact.tsx**: Actualiza las URLs de GitHub, Kaggle y LinkedIn
- **Hero.tsx**: Ajusta el titular y la descripción
- **About.tsx**: Edita tu bio, habilidades y formación
- **Header.tsx**: Ajusta el nombre visible en la cabecera

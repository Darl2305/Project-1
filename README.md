# Portafolio Personal - Data & ML

Portafolio con tema oscuro/claro, sección About me, frase inspiradora, Personal Projects y links de contacto (GitHub, Kaggle, LinkedIn).

## Cómo ejecutar

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Ejecutar en desarrollo:
   ```bash
   npm run dev
   ```

3. Abrir [http://localhost:3000](http://localhost:3000)

## Deploy en Vercel

1. Sube el proyecto a GitHub
2. Conecta el repo en [vercel.com](https://vercel.com)
3. Vercel detectará Next.js automáticamente

## Personalización

- **Contact.tsx**: Actualiza las URLs de GitHub, Kaggle y LinkedIn
- **Hero.tsx**: Cambia la frase inspiradora, descripción y tags
- **About.tsx**: Edita tu bio y skills
- **Header.tsx**: Cambia "Portfolio" por tu nombre o iniciales
